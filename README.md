# DVSA Vulnerability Discovery and Remediation

This project is part of the ICS344 Information Security course.  
It focuses on identifying, exploiting, and fixing vulnerabilities in the DVSA.


## Project Overview

- Platform: AWS Serverless (Lambda, API Gateway, DynamoDB)
- Focus: Security vulnerabilities in serverless applications
- Tasks:
  - Reproduce vulnerabilities
  - Analyze root causes
  - Apply fixes
  - Verify mitigation
